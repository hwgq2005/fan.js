Fan.js 文档
====================
插件说明：
-
此插件支持`移动端` 、`PC端`，不需要引入第三方支持库，支持IE9及以上。
DOM结构
-
	<div class="menu-box">
	    <a href="javascript:;" class="menu-btn" id="menu-btn">+</a>
	    <div class="menu-wrapper">
	    	<ul>
	    		<li><a href="#">文件</a></li>
	    		<li><a href="#">日程</a></li>
	    		<li><a href="#">管理</a></li>
	    		<li><a href="#">任务</a></li>
	    		<li><a href="#">团队</a></li>
	    		<li><a href="#">资源</a></li>
	    		<li><a href="#">圈子</a></li>
	    		<li><a href="#">工具</a></li>
	    		<li><a href="#">招聘</a></li>
	    	</ul>
	    </div>
	</div>

引入
-
	<link href="css/fan.css" rel="stylesheet">
	<script src="js/fan.js"></script>
JS调用
-
	Fan.init('#menu-btn');
  

该插件仅供学习，不断完善中。
